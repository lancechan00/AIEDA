# 模型后端

第一阶段只讨论同一任务 `LocalRouteChoice-Lite`。

## 角色划分

### `tiny_baseline`

正式训练后端。

用途：

- 验证数据集是否可学。
- 验证 `train -> val -> eval` 最小闭环是否成立。
- 作为后续实验回归基线。

### `DeepSeek-VL`

实验后端接口占位。

当前状态：

- 不进入正式训练主链。
- 不假装为可训练主干。
- 保留统一输入输出接口，便于后续接入 prompt / inference baseline。

### `Janus`

实验后端接口占位。

当前状态：

- 不进入正式训练主链。
- 不承担第一阶段训练可行性验证。
- 当前只保留同任务实验接口。

## 第一阶段结论标准

只有在 `tiny_baseline` 已经稳定跑通之后，才继续讨论：

- 是否接入真正的 `DeepSeek-VL` 推理实验。
- 是否接入真正的 `Janus` 推理实验。
- 是否值得继续投入大模型后端。
